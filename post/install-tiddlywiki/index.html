Install Tiddlywiki

@Can Yellow · Nov 21, 2018 · 1 min read

tiddlywiki是一个个人的wiki系统，这里就介绍我在ubuntu18上安装tiddlywiki5的经过。

tiddlywiki5的官网地址如下：https://tiddlywiki.com/，tiddlywiki5是在node.js的基础上开发的。所以tiddlywiki既可以配置成服务器上使用，也可以配置成桌面传统使用，tiddlywiki团队甚至退出了一款叫做tiddlyDesktop的桌面APP，讲起来也就是现在流行的使用electron或者NW.JS来利用web技术封装的桌面应用。当然只要系统上有node.js环境，不管是windows还是linux亦或者MacOS都不是问题。我选择的就是在node.js环境中安装tiddlywiki5.

安装nvm

nvm是一个node.js版本的管理工具，提供对于node.js与其模块的管理功能，因此安装node.js之前先安装nvm，再通过nvm安装node.js。nvm托管在github上，官方详细的安装指南在这里。具体的安装过程也参考了其他博客1 首先，我确定的nvm的安装目录是~/nodejs,所以先创建并进入该目录

$ cd ~
$ mkdir nodejs
$ cd nodejs

下载nvm源码

git clone https://github.com/creationix/nvm.git

配置终端启动时自动执行source ~/nodejs/nvm/.nvm/nvm.sh，在~/.bashrc文件末尾追加如下内容

# link to nvm
source /home/canyellow/nodejs/nvm/.nvm/nvm.sh

重启终端，在终端中输入

$ nvm

若显示command not found说明没有正确配置安装nvm。 若输出如下内容，说明nvm已经在系统中正确安装。

Node Version Manager

Note: <version> refers to any version-like string nvm understands. This includes:
...
...
Note:
  to remove, delete, or uninstall nvm - just remove the `$NVM_DIR` folder (usually `~/.nvm`)

安装node.js

如果已经正确安装nvm，则安装node.js的过程一个命令就可以搞定。

$ NVM_NODEJS_ORG_MIRROR=https://npm.taobao.org/mirrors/node
$ nvm install --lts //--lts指定安装当前最新稳定版

node.js最终默认安装于~/nodejs/nvm/.nvm/versions/node/v10.13.0路径

安装tiddlywiki5

同样在node.js已经安装的情况下，安装tiddlywiki5也很简单，一条代码搞定。

$ sudo npm install -g tiddlywiki

确认是否安装OK，可以在命令行输入下面的命令。

$ tiddlywiki --version
> 5.1.17

如果出现上面类似的版本号，说明安装完成，如果提示command not found，说明安装遇到问题。

配置网站

我的笔记目录设置在~/mynote/mynote/。为此，先创建并进入该目录。

$ cd ~
$ mkdir mynote
$ cd ~/mynote
$ mkdir mynote
$ cd mynote

将该目录初始化为tiddlywiki的文件夹

$ tiddlywiki --init

启动tiddlywiki服务使用下面的命令

$ tiddlywiki --server

输入后，浏览器访问http://localhost:8080，就可以访问tiddlywiki的页面文件了。 如果需要将tiddlywiki设置成开机启动的话，在/etc中建立名为start_tiddlywiki.sh的脚本，里面的内容如下：

#!/bin/bash
# run tiddlywiki server
source /home/canyellow/nodejs/nvm/.nvm/nvm.sh
cd /home/canyellow/mynote/mynote/
nohup tiddlywiki --server>tiddlywiki.log 2>&1 &

然后向/etc/rc.local文件中添加如下代码，开机自动运行上面的脚本文件即可。

# run tiddlywiki server
/etc/start_tiddlywiki.sh

添加markdown与katex支持

完成tiddlywiki并初始化tiddlywiki文件夹后，浏览器访问localhost:8080在打开的页面里点击侧边栏的设置按钮，根据个人喜好更改设置，然后安装插件增加中文支持、markdown语法解析、数学公式解析功能。 在tiddlywiki主页按如下操作：在侧边栏点击设置→在出现的页面中点击插件选项→点击更多插件打开插件浏览下载对话框→在下拉列表中选择“KaTeX: mathematical typography”、“Markdown parser”、“Chinese (Simplified)”（见下方图片）分别点击安装→在侧边栏点击保存变更→刷新→在设置里选择语言为简体中文附图。

安装tiddlyMap

tiddlymap是tiddlywiki上的一块思维导图插件，安装过程如官方指南说明： 1. 点击TiddlyMap，TW5-Vis.js，TW5-HotZone，TW5-TopStoryView四个项目对应的项目链接，下载zip格式的源文件 2. 将四个项目解压，将解压出来的四个文件夹的./dist目录合并，并复制到tiddlywikiwiki安装目录（上面nodejs安装目录中的node_module文件夹下）中的tiddlywiki/plugins/ 然后在建立网站项目（刚刚使用tiddlywiki –init命令）的文件夹中打开tiddlywiki.info文件，按照原有格式，添加下面的配置：

  "plugins": [
    "felixhayashi/tiddlymap",
    "felixhayashi/vis",
    "felixhayashi/hotzone",
    "felixhayashi/topstoryview"
  ]

移动版方案

手机上最简单的方案是下载安装AndTidWiki软件，然后将tiddlywiki5的空模板从下载地址下载后放到手机的andtidwiki文件夹中。
